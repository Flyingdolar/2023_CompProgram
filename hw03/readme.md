# HW03 作業

> 40768033T 鄭翔元

## 作業內容

### 1. 簡易加密器

- 設定一個輸入與一個輸出的檔案
- 演算法使用 XOR 方式加密並輸出到輸出檔案
- 同樣，使用 XOR 還原輸出檔案

### 2. ZIP 檔案資料夾解析器

- 輸入一個 ZIP 檔案
- 解析 ZIP 檔案中的資料夾與檔案
- 可以設定排列順序：檔案名稱正序、檔案名稱反序

### 3. BMP 打馬賽克

- 輸入一個 BMP 檔案、一個輸出的 BMP 檔案
- 設定打馬賽克的位置大小、網點大小

### 4. 金庸群俠轉：金手指

- 輸入一個金庸群俠傳的存檔
- 依據 Help 內容，設定要修改的項目
- 輸出修改後的存檔

### 5. MIDI 音樂檔案生成器

- 輸入一個 .txt 的樂譜檔案
- 依據樂譜檔案，生成一個 MIDI 檔案
